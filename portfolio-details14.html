Project Story

I worked on a project to develop a monitoring system for two UPS units, focusing on key parameters to ensure efficient performance tracking. The user requested a display interface that included both gauges and trend graphs for visualizing the 3-phase power metrics. The parameters being monitored were:
  • UPS' Status
  • Voltage
  • Current
  • Frequency
  • Capacity
Additionally, the system incorporated a status indicator to show whether the UPS units were on or off. Communication between the monitoring system and the UPS units was established using the Modbus TCP/IP protocol, ensuring reliable and real-time data transmission. The goal of this project was to create a user-friendly and interactive monitoring solution for real-time tracking of the UPS units' performance and operational status.
