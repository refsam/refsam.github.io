Project Story

This is the final project of the Digital Talent Academy training program in the UI/UX Designer field, which is organized by the Ministry of Communication and Information Technology (Kominfo). The final project provides various challenges, one of which is to create a design solution so that we can with the following objectives: “Provide donors with the latest news (campaign updates) to make their donation experience more emotionally engaging and increase retention rates”

Design Output

1. Thanks Card

The feature is aimed to express gratitude to donors on the Kitabisa application, which aims to increase the empathy and emotional response of donors after donating. This thanks card will appear or be sent automatically to the user after the donation payment transaction is declared successful.

2. Personal Message

The Personal Message feature allows campaign creators to provide personal messages to donors, such as expressing gratitude and providing the latest updates, with the aim of strengthening the emotional bond between donors and donation organizers. This feature can be accessed on the inbox page and in the message section.

3. Latest Information

A special page that displays the latest information from the campaign donated by donors. This feature aims to give donors the opportunity to see the latest information or donate again to the campaign. This feature can be accessed on the inbox page and in the message section.
